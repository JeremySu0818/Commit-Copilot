# Informazioni sull'aggiornamento di Commit Copilot

## Novità della versione 1.13.0

- Aggiunta l'opzione "Lingua del messaggio di commit" nella pagina delle impostazioni, indipendente dalla lingua dell'interfaccia dell'estensione.
- Aggiunta l'opzione "Generazione ibrida" nella pagina delle impostazioni per utilizzare l'input SCM corrente come bozza di riferimento.
- Aggiunta l'opzione del prefisso Gitmoji (`includeGitmoji`) nelle impostazioni di generazione per aggiungere emoji ai messaggi di commit generati.
- Aggiunta un'icona GitHub nella vista principale che collega direttamente al repository dell'estensione.
- Aggiunto un pulsante nella vista principale per aprire la pagina "Informazioni" (Info/About), che mostra i dettagli della versione e dell'autore.
- Aggiunta una scheda con le informazioni di aggiornamento, visualizzata al primo avvio dopo un aggiornamento dell'estensione.
- Architettura dei prompt completamente localizzata, con supporto per tutte le 20 lingue.
- Rinominato il comando delle impostazioni in `openSettings` e migliorata la coerenza dell'interfaccia utente.
