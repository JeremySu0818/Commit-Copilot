# Commit Copilot Update-Informationen

## Was ist neu in Version 1.13.0

- Option "Sprache der Commit-Nachrichten" auf der Einstellungsseite hinzugefügt, unabhängig von der Sprache der Erweiterungs-UI.
- Option "Hybride Generierung" auf der Einstellungsseite hinzugefügt, um die aktuelle SCM-Eingabe als Referenzentwurf zu verwenden.
- Gitmoji-Präfix-Option (`includeGitmoji`) in den Generierungseinstellungen hinzugefügt, um generierten Commit-Nachrichten Emojis hinzuzufügen.
- GitHub-Symbol in der Hauptansicht hinzugefügt, das direkt zum Repository der Erweiterung verlinkt.
- Schaltfläche in der Hauptansicht hinzugefügt, um die "Über"-Seite (Info/About) mit Versions- und Autorendetails zu öffnen.
- Ein Tab für Update-Informationen wurde hinzugefügt, der beim ersten Start nach einem Update der Erweiterung angezeigt wird.
- Die Prompt-Architektur wurde vollständig lokalisiert und unterstützt alle 20 Sprachen.
- Der Einstellungsbefehl wurde in `openSettings` umbenannt und die UI-Konsistenz verbessert.
