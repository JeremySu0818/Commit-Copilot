# Thông tin cập nhật Commit Copilot

## Tính năng mới trong Phiên bản 1.14.0

- Hỗ trợ chế độ proxy của Ollama: Giới thiệu giao thức công cụ tác nhân văn bản để loại bỏ cơ chế dự phòng Direct Diff bắt buộc, và hỗ trợ hủy tiến trình tải xuống (Pull) mô hình Ollama khi hủy tạo tin nhắn.
- Hỗ trợ nhà cung cấp tùy chỉnh Anthropic: Cho phép thiết lập điểm cuối tùy chỉnh theo định dạng Anthropic API, định cấu hình tối đa token đầu ra, tối ưu hóa thứ tự nhập các trường mới và tự động di chuyển các cấu hình cũ.
- Mô-đun hóa kiến trúc cốt lõi: Chia nhỏ các thành phần cốt lõi như điều phối tạo tin nhắn, thao tác Git, quản lý mô hình và giao thức webview thành các mô-đun độc lập, đồng thời mô-đun hóa các gợi ý (prompt) ngôn ngữ để cải thiện hiệu suất tải.
- Đơn giản hóa tên hiển thị nhà cung cấp: Sửa nhãn nhà cung cấp tích hợp sẵn thành các tên gọn gàng hơn.
- Sửa nhãn ngôn ngữ giao diện (UI): Điều chỉnh nhãn hành động của trình chọn mô hình từ "Thêm mô hình" thành "Quản lý mô hình..." để phù hợp hơn với màn hình tính năng.
- Cập nhật và tối ưu hóa tài liệu README.md cùng các ví dụ cấu hình.
