# Commit Copilot अपडेट जानकारी

## संस्करण 1.14.0 में नया क्या है

- Ollama प्रॉक्सी मोड का समर्थन: अनिवार्य Direct Diff फ़ॉलबैक तंत्र को हटाने के लिए टेक्स्ट एजेंट टूल प्रोटोकॉल पेश किया गया, और जनरेशन रद्द करने पर Ollama मॉडल डाउनलोड (Pull) प्रगति को रोकने का समर्थन किया गया।
- Anthropic कस्टम प्रदाता का समर्थन: Anthropic API प्रारूप में कस्टम एंडपॉइंट सेट करने, अधिकतम आउटपुट टोकन कॉन्फ़िगर करने की अनुमति दी गई, साथ ही नए फ़ील्ड के इनपुट क्रम को अनुकूलित किया गया और पुरानी सेटिंग्स को स्वचालित रूप से माइग्रेट किया गया।
- कोर आर्किटेक्चर का मॉड्यूलराइजेशन: जनरेशन ऑर्केस्ट्रेशन, Git संचालन, मॉडल प्रबंधन और webview प्रोटोकॉल जैसे कोर घटकों को स्वतंत्र मॉड्यूल में विभाजित किया गया, और लोड प्रदर्शन को बेहतर बनाने के लिए भाषा प्रॉम्ट को मॉड्यूलराइज़ किया गया।
- प्रदाता के प्रदर्शन नामों का सरलीकरण: इन-बिल्ट प्रदाता लेबल को अधिक साफ नामों में सुधारा गया।
- UI भाषा लेबल में सुधार: मॉडल चयनकर्ता के एक्शन लेबल को "मॉडल जोड़ें" से बदलकर "मॉडल प्रबंधित करें..." किया गया ताकि यह कार्य स्क्रीन के साथ बेहतर ढंग से मेल खा सके।
- README.md दस्तावेज़ीकरण और कॉन्फ़िगरेशन उदाहरणों को अपडेट और अनुकूलित किया गया।
