# Информация об обновлении Commit Copilot

## Что нового в версии 1.13.0

- На страницу настроек добавлена опция «Язык сообщений коммита», не зависящая от языка интерфейса расширения.
- На страницу настроек добавлена опция «Гибридная генерация», позволяющая использовать текущий ввод SCM в качестве черновика.
- В настройки генерации добавлена опция префикса Gitmoji (`includeGitmoji`) для добавления эмодзи в сгенерированные сообщения коммитов.
- На главный экран добавлена иконка GitHub, ведущая непосредственно в репозиторий расширения.
- На главный экран добавлена кнопка для открытия страницы «О программе» (Info/About), отображающей информацию о версии и авторе.
- Добавлена вкладка с информацией об обновлении, которая отображается при первом запуске после обновления расширения.
- Архитектура промптов полностью локализована с поддержкой всех 20 языков.
- Команда настроек переименована в `openSettings`, улучшена согласованность пользовательского интерфейса.
