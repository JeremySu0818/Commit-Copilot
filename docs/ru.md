# Информация об обновлении Commit Copilot

## Что нового в версии 1.14.0

- Поддержка прокси-режима Ollama: добавлены протоколы текстовых агентов для устранения принудительного использования резервного механизма Direct Diff, а также добавлена возможность прерывания загрузки (Pull) модели Ollama при отмене генерации.
- Поддержка кастомных провайдеров Anthropic: добавлена возможность настройки пользовательских эндпоинтов в формате API Anthropic, настройки максимального количества выходных токенов, оптимизирован порядок ввода новых полей и реализован автоматический перенос старых настроек.
- Модулирование базовой архитектуры: основные компоненты, такие как оркестрация генерации, операции Git, управление моделями и протоколы webview, разделены на независимые модули, а языковые промпты модулированы для повышения скорости загрузки.
- Упрощение отображаемых имен провайдеров: метки встроенных провайдеров изменены на более лаконичные.
- Исправление языковых меток интерфейса: метка действия в селекторе моделей изменена с «Добавить модель» на «Управление моделями...» для большего соответствия экрану функции.
- Обновление и оптимизация документации README.md и примеров конфигурации.
