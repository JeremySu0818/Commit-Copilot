# Commit Copilot 更新情報

## バージョン 1.13.0 の新機能

- 設定ページに、拡張機能のUI言語とは独立して設定できる「コミットメッセージの言語」オプションを追加しました。
- 設定ページに、現在の SCM 入力テキストを下書きとして参照する「ハイブリッド生成」オプションを追加しました。
- 生成設定に、生成されるコミットメッセージに絵文字を追加できる Gitmoji プレフィックスオプション（`includeGitmoji`）を追加しました。
- メインビューに、拡張機能のリポジトリに直接リンクする GitHub アイコンを追加しました。
- メインビューに、バージョンと作成者情報を表示する「About（バージョン情報）」ページを開くボタンを追加しました。
- 更新情報タブを追加し、拡張機能の更新後の初回起動時に表示するようにしました。
- プロンプトアーキテクチャを完全にローカライズし、20言語すべてをサポートしました。
- 設定コマンドの名前を `openSettings` に変更し、UI の一貫性を向上させました。
