# Commit Copilot 更新情報

## バージョン 1.14.0 の新機能

- Ollama のプロキシモードに対応：テキストエージェントツールプロトコルを導入して強制的な Direct Diff フォールバックメカニズムを廃止し、生成キャンセル時に Ollama モデルのダウンロード（Pull）進行状況を中止できるようにしました。
- Anthropic カスタムプロバイダーに対応：Anthropic API 形式のカスタムエンドポイントの設定、最大出力トークンの構成を可能にし、新しいフィールドの入力順序の最適化および旧バージョンの設定の自動移行を行いました。
- コアアーキテクチャのモジュール化：生成のオーケストレーション、Git 操作、モデル管理、webview プロトコルなどのコア機能を独立したモジュールに分割し、言語プロンプトをモジュール化してロードパフォーマンスを向上させました。
- プロバイダー表示名の簡素化：組み込みプロバイダーのラベルをよりクリーンな名称に修正しました。
- UI 言語ラベルの修正：モデルセレクターのアクションラベルを「モデルを追加」から「モデルを管理...」に修正し、実際の機能画面と一致させました。
- README.md 説明ドキュメントと設定例の更新および最適化。
