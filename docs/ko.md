# Commit Copilot 업데이트 정보

## 버전 1.13.0의 새로운 기능

- 설정 페이지에 확장 프로그램 UI 언어와 독립적으로 설정할 수 있는 "커밋 메시지 언어" 옵션을 추가했습니다.
- 설정 페이지에 현재 SCM 입력 내용을 초안 참조로 사용할 수 있는 "하이브리드 생성" 옵션을 추가했습니다.
- 생성 설정에 생성된 커밋 메시지에 이모지를 추가할 수 있는 Gitmoji 접두사 옵션(`includeGitmoji`)을 추가했습니다.
- 메인 뷰에 확장 프로그램 리포지토리로 직접 연결되는 GitHub 아이콘을 추가했습니다.
- 메인 뷰에 버전 및 작성자 정보를 표시하는 "About(정보)" 페이지를 여는 버튼을 추가했습니다.
- 업데이트 정보 탭을 추가하여 확장 프로그램 업데이트 후 첫 실행 시 표시되도록 했습니다.
- 프롬프트 아키텍처를 완전히 현지화하여 20개 언어 모두를 지원합니다.
- 설정 명령을 `openSettings`로 이름 변경하고 UI 일관성을 개선했습니다.
