# Commit Copilot 업데이트 정보

## 버전 1.14.0의 새로운 기능

- Ollama 프록시 모드 지원: 텍스트 에이전트 툴 프로토콜을 도입하여 강제 Direct Diff 대체 메커니즘을 제거하고, 생성 취소 시 Ollama 모델 다운로드(Pull) 진행 상황을 중단할 수 있도록 지원합니다.
- Anthropic 사용자 정의 제공자 지원: Anthropic API 형식의 사용자 정의 엔드포인트를 설정하고 최대 출력 토큰을 구성할 수 있게 했으며, 새 필드의 입력 순서 최적화 및 기존 설정의 자동 마이그레이션을 지원합니다.
- 핵심 아키텍처 모듈화: 생성 오케스트레이션, Git 작업, 모델 관리, webview 프로토콜 등 핵심 구성 요소를 독립 모듈로 분할하고, 언어 프롬프트를 모듈화하여 로딩 성능을 향상했습니다.
- 제공자 표시 이름 단순화: 기본 제공자의 라벨을 더 깔끔한 이름으로 수정했습니다.
- UI 언어 라벨 수정: 모델 선택기 작업 라벨을 "모델 추가"에서 "모델 관리..."로 수정하여 기능 화면과 일치시켰습니다.
- README.md 설명서 및 설정 예시 업데이트 및 최적화.
