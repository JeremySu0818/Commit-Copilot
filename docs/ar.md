# معلومات تحديث Commit Copilot

## الميزات الجديدة في الإصدار 1.14.0

- دعم وضع الوكيل لـ Ollama: تم إدخال بروتوكولات أدوات وكيل النصوص لإلغاء آلية Direct Diff الاحتياطية الإلزامية، ودعم إيقاف تقدم تنزيل (Pull) نموذج Ollama عند إلغاء التوليد.
- دعم مزودي الخدمة المخصصين لـ Anthropic: السماح بتعيين نقاط نهاية مخصصة بتنسيق Anthropic API، وتهيئة الحد الأقصى لرموز المخرجات (Tokens)، وتحسين ترتيب إدخال الحقول الجديدة مع الترحيل التلقائي للإعدادات القديمة.
- معمارية أساسية نموذجية (Modular): تقسيم المكونات الأساسية مثل تنسيق التوليد، وعمليات Git، وإدارة النماذج، وبروتوكولات webview إلى وحدات مستقلة، وتمكين نمذجة مطالبات اللغة (Prompts) لتحسين أداء التحميل.
- تبسيط أسماء عرض مزودي الخدمة: تصحيح تسميات مزودي الخدمة المدمجين إلى أسماء أكثر بساطة ونقاء.
- تصحيح تسميات لغة واجهة المستخدم: تصحيح تسمية إجراء محدد النماذج من "إضافة نموذج" إلى "إدارة النماذج..." لتتوافق بشكل أفضل مع شاشة الوظيفة.
- تحديث وتحسين وثائق README.md وأمثلة الإعداد.
