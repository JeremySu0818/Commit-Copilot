# معلومات تحديث Commit Copilot

## ما الجديد في الإصدار 1.13.0

- تمت إضافة خيار "لغة رسالة الالتزام" في صفحة الإعدادات، بشكل مستقل عن لغة واجهة المستخدم للإضافة.
- تمت إضافة خيار "التوليد الهجين" في صفحة الإعدادات لاستخدام إدخال SCM الحالي كمسودة مرجعية.
- تمت إضافة خيار بادئة Gitmoji (`includeGitmoji`) في إعدادات التوليد لإضافة رموز تعبيرية إلى رسائل الالتزام المُنشأة.
- تمت إضافة أيقونة GitHub في العرض الرئيسي ترتبط مباشرة بمستودع الإضافة.
- تمت إضافة زر في العرض الرئيسي لفتح صفحة "حول" (Info/About)، والتي تعرض تفاصيل الإصدار واسم المؤلف.
- تمت إضافة علامة تبويب معلومات التحديث، والتي تظهر عند التشغيل الأول بعد تحديث الإضافة.
- تمت ترجمة بنية التلقين (Prompt) بالكامل، مع دعم لجميع اللغات العشرين.
- تمت إعادة تسمية أمر الإعدادات إلى `openSettings` وتم تحسين اتساق واجهة المستخدم.
