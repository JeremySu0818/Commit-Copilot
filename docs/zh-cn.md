# Commit Copilot 更新信息

## 版本 1.13.0 的新功能

- 在设置页面中新增“提交信息语言”选项，可独立于扩展界面语言进行设置。
- 在设置页面中新增“混合式生成”选项，可将当前 SCM 输入框的内容作为草稿参考。
- 在生成设置中新增 Gitmoji 前缀选项（`includeGitmoji`），可为生成的 commit 提交信息加入表情符号。
- 在主视图中新增 GitHub 图标直连 Extension 仓库。
- 在主视图中新增按钮打开“关于（Info/About）”页面，显示版本与作者信息。
- 新增更新信息分页，在扩展更新后的首次启动时显示。
- 完整本地化 Prompt 架构，支持所有 20 种语言。
- 将设置命令重命名为 `openSettings`，并改善了界面的一致性。
