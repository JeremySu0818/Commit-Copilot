# Commit Copilot 更新信息

## 版本 1.14.0 的新功能

- 支持 Ollama 的代理模式：引入文本代理工具协议以移除强制的 Direct Diff 备用机制，并支持在取消生成时中止 Ollama 模型的下载（Pull）进度。
- 支持 Anthropic 自定义服务商：允许设置 Anthropic API 格式的自定义端点、配置最大输出 Token，并优化了新增字段的输入顺序与自动迁移旧版设置。
- 核心架构模块化：将生成编排、Git 操作、模型管理与 webview 协议等核心拆分为独立模块，并将语系 Prompt 模块化以提升加载性能。
- 简化服务商显示名称：将内置服务商标签修正为更干净的名称。
- 修正 UI 语系标签：将模型选择器动作标签从“新增模型”修正为“管理模型...”，使其更符合功能画面。
- 更新与优化 README.md 说明文档与配置示例。
