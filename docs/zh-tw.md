# Commit Copilot 更新資訊

## 版本 1.13.0 的新功能

- 在設定頁面中新增「Commit 訊息語言」選項，可獨立於擴充套件介面語言進行設定。
- 在設定頁面中新增「混合式生成」選項，可將目前 SCM 輸入框的內容作為草稿參考。
- 在產生設定中新增 Gitmoji 前綴選項（`includeGitmoji`），可為產生的 commit 訊息加入表情符號。
- 在主畫面中新增 GitHub 圖示直連 Extension 倉庫
- 在主畫面中新增按鈕打開「關於（Info/About）」頁面，顯示版本與作者資訊。
- 新增更新資訊分頁，在擴充套件更新後的首次啟動時顯示。
- 完整在地化 Prompt 架構，支援所有 20 種語言。
- 將設定指令重新命名為 `openSettings`，並改善介面一致性。
