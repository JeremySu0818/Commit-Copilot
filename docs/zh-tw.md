# Commit Copilot 更新資訊

## 版本 1.14.0 的新功能

- 支援 Ollama 的代理模式：引入文字代理工具協定以移除強制的 Direct Diff 備用機制，並支援在取消生成時中止 Ollama 模型的下載（Pull）進度。
- 支援 Anthropic 自訂服務商：允許設定 Anthropic API 格式的自訂端點、配置最大輸出 Token，並優化了新增欄位的輸入順序與自動遷移舊版設定。
- 核心架構模組化：將生成編排、Git 操作、模型管理與 webview 協定等核心拆分為獨立模組，並將語系 Prompt 模組化以提升載入效能。
- 簡化服務商顯示名稱：將內建服務商標籤修正為更乾淨的名稱。
- 修正 UI 語系標籤：將模型選擇器動作標籤從「新增模型」修正為「管理模型...」，使其更符合功能畫面。
- 更新與優化 README.md 說明文件與設定範例。
