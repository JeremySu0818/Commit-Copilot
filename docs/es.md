# Información de actualización de Commit Copilot

## Novedades de la versión 1.13.0

- Se agregó la opción "Idioma del mensaje de commit" en la página de configuración, independiente del idioma de la interfaz de la extensión.
- Se agregó la opción "Generación híbrida" en la página de configuración para usar la entrada actual del SCM como borrador de referencia.
- Se agregó la opción de prefijo Gitmoji (`includeGitmoji`) en la configuración de generación para agregar emojis a los mensajes de commit generados.
- Se agregó un ícono de GitHub en la vista principal que enlaza directamente al repositorio de la extensión.
- Se agregó un botón en la vista principal para abrir la página "Acerca de" (Info/About), que muestra detalles de la versión y del autor.
- Se agregó una pestaña de información de actualización, que se muestra en el primer inicio después de actualizar la extensión.
- Se localizó completamente la arquitectura de prompts, con soporte para los 20 idiomas.
- Se renombró el comando de configuración a `openSettings` y se mejoró la consistencia de la interfaz de usuario.
